# SEO Fixes ‚Äî 2026-05-21

## Changes

### P0 ‚Äî Removed `X-Robots-Tag: noindex` from production
- **`next.config.ts`**: Removed `async headers()` block that sent `X-Robots-Tag: noindex` globally.
  This header was overriding meta robots tags and blocking all pages from being indexed on production.

### P1 ‚Äî Fixed canonical + hreflang per page
- **New file `src/lib/metadata.ts`**: `localeAlternates(lang, path)` helper generating:
  - `canonical` URL for the specific page
  - `hreflang` alternates (`fr-FR`, `en-US`, `x-default`)
- **`[lang]/layout.tsx`**: Removed `alternates` block ‚Äî it was setting ALL pages' canonical to `/${lang}/`
- **19 page.tsx files**: Each now sets its own canonical + hreflang via `...localeAlternates(lang, "/path")`

### P1B ‚Äî Refactored `insights/` to server wrapper
- Renamed `insights/page.tsx` ‚Üí `insights/InsightsClient.tsx` (client component)
- Created new `insights/page.tsx` (server component) with `generateMetadata` including canonical + hreflang
- `/insights/` now properly indexed by Google with correct metadata

### P1C ‚Äî Noindexed `usecases/`
- Renamed `usecases/page.tsx` ‚Üí `usecases/UsecasesClient.tsx` (client component)
- Created new `usecases/page.tsx` (server component) with `robots: { index: false }`

### P2 ‚Äî HTTP‚ÜíHTTPS + non-WWW‚ÜíWWW redirects
- **`public/.htaccess`**: Added canonical redirect rule (single combined RewriteRule):
  ```apache
  RewriteCond %{HTTPS} off [OR]
  RewriteCond %{HTTP_HOST} !^www\. [NC]
  RewriteRule ^(.*)$ https://www.mentivis.com/$1 [R=301,L]
  ```

### P3 ‚Äî Deleted dead pages
- Deleted `src/app/[lang]/homepage-old/` ‚Äî old unused homepage
- Deleted `src/app/[lang]/mentivissolutions/` ‚Äî replaced by `/solutions` + external `/mentivis-solutions` iframe
  - `.htaccess` 301 redirects for `mentivissolutions` ‚Üí homepage remain intact

### Cleanup
- **`src/lib/contact-url.ts`**: Removed dead `mentivissolutions` entry
- **`scripts/gen-llms-txt.js`**: Removed `mentivissolutions` references, updated `skillpath` ‚Üí `mentivisos`, added `ambassadors`
