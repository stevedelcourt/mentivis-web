## SEO Infrastructure

### Canonical + hreflang per page
- **New helper** `src/lib/metadata.ts` — `localeAlternates(lang, path)` generates correct canonical + hreflang + x-default for any route
- Removed incorrect global `alternates` from `[lang]/layout.tsx` (was setting all pages' canonical to `/${lang}/`)
- All 19 public pages now set their own canonical + hreflang via `...localeAlternates(lang, "/path")` in `generateMetadata`

### HTTP→HTTPS + WWW redirects
- **`.htaccess`** — added canonical redirect: HTTP→HTTPS + non-WWW→WWW, scoped to `mentivis.com` only (preserves sc3 dev access)
- **Root → `/fr/` 301** — eliminated duplicate homepage indexing (like Air France)

### Zombie pages removed
- Deleted `homepage-old/` — dead old homepage
- Deleted `mentivissolutions/` — replaced by `/solutions` + external iframe link
- `usecases/` — `robots: noindex` via server wrapper

### Fixed articles infrastructure
- `insights.ts` re-exports preserved across auto-regeneration
- `insights/` listing refactored to server wrapper (properly indexed)
- Article CTA simplified to "Contact"
- Stale slug fix in featured-insights.json
